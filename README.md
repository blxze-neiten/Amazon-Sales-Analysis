# Amazon Sales Analysis

## Project Overview
This project analyzes Amazon sales data using Python and Microsoft Excel. The objective is to explore sales trends, customer behavior, product performance, and payment methods through data analysis and visualization.

## Dataset
- Dataset: `amazon.xlsx`

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab
- Microsoft Excel

## Analysis Performed
- Data cleaning
- Dataset information (`info()`)
- Descriptive statistics (`describe()`)
- Total Sales
- Total Orders
- Average Sales
- Maximum Quantity
- Revenue by State
- Orders by Payment Mode
- Top 5 Customers
- Charts and visualizations

## Dashboard
An interactive Excel dashboard is included in:
- `Amazon Dashboard.xlsx`

## Files
- `amazon.xlsx` – Dataset
- `Amazon_Sales_Analysis.ipynb` – Python analysis notebook
- `Amazon Dashboard.xlsx` – Excel Dashboard
- `README.md` – Project documentation
